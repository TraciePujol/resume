PLNR U

Web Application

In Development

Project Background:

The idea for PLNR U initially stemmed from a concept explored in a previous class. It later evolved into a passion project and was further developed as a class project.

Description:

Developing a project planning utility software designed to streamline project management tasks and enhance team collaboration.

Role:

Lead Developer

Versions:

1. Class Project (ISDS 4125)

2. Passion Project (Personal)

Class Project Version:

The Class Project Version of PLNR U was developed as a Minimum Viable Product (MVP) during the course. It served as an introduction to the project's core concepts and features.

Class Project Version Timeline:

Development started: March 2023 - May 2023; August 2023 - October 2023

Currently in the design phase, working on detailed system and database design. Development/implementation phase scheduled to start mid-September

Passion Project Version:

The Passion Project Version of PLNR U is my independent initiative. It builds upon the concepts explored in the class project and is intended for a more comprehensive implementation.

Passion Project Version Timeline:
